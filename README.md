# CursoEmVídeo Python Exercises

This repository contains exercises and challenges for learning Python from the CursoEmVídeo course. It includes practice files and small scripts organized in folders such as `desafios`, `exercicios`, and `scripts`.

Purpose
- Track progress while following the CursoEmVídeo Python lessons.
- Keep simple examples and solved exercises for reference.

Usage
- Run individual Python files with Python 3 (for example: `python3 desafios/desafio_001.py`).
